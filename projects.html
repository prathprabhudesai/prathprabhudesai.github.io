  • Water Detection & Segmentation - SVM, NN, PCA, Clustering ( Code | Project Report | )

    • Implementated the Support Vector Machines, Neural Networks, Principal Component ANalysis and Clustering algorithms to detect water in a topographical area, segment that area and classify as flood or non-flood region.
    Features used: Mean and Variance of Hue and Saturation
    Neural Networks: MATLAB toolbox for ANN is used for training and testing purpose.
    Support Vector Machines: Since the data distribution is non-linear, radial basis function is used to get the separating plane. Sequential Minimal Optimization is used to solve the optimization problem.
    Principal Component Analysis: KL transform is performed on the data and projected onto minor eigen vector.
    Clustering: K-means Clustering is used.

    Result of Multiscale Segmentation using Support Vector Machines

  • Statistical Modelling of Fish Population
    ( Project Report | )

    • Echo SONAR is used for the determination of fish population which used sinusoidal sound pulses with Rayleigh Distributed amplitudes. The school of fish was found to be Normally Distributed (Gaussian) verifying the Central Limit Theorem. The process was modelled using fish's position, orientation and motion as random variables.

  • Open Stream Water Velocity Profiler using Image Processing ( Project Report | )

    • This system was developed for open-stream water velocity profiling using Image Processing algorithms influenced by motion tracking and Particle Image Velocimetry. Research was carried out on Mumbai Port Model to tackle the safety issues regarding over-flooding in the area of Navi Mumbai International Airport. Then existing hardware technologies mainly included Vertical Acoustic Doppler Current Profiler (ADCP) sensors which are costly and fail to achieve directional profiling. This non-contact technology provided a low cost and accurate solution overcoming the existing drawbacks.
